Jan 4 2004
Paweł Olszta works on IL code generation using System.Reflection.Emit to replace our current approach with C# compiler in the middle. He was also working on Nemerle type export/import using extended attributes. Michal Moskal has implemented tail recursion elimination, some limited type inference and rewritten parser in Nemerle. Kamil Skalski has been working on macros -- several macros are fully functional now (printf, for, using). He also rewrite lexer in Nemerle. Compiler contains no C# sources any more. Dawid Duda has written few standard library modules.
